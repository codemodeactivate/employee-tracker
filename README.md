## Installation

Make sure to have node installed. You can type:

    ```

    npm i

    ```

Switch to your MySQL shell and type:

```
source db/schema.sql
source db/seeds.sql
```

Once your dependencies are added, just type

```
node index.js
```

to run from your terminal.
